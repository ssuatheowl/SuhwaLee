Change My Number

01Project Summary

Contribution

UX design

UX writing

One would say it should be possible to change a phone number of an insurance account, but it wasn't the case for Carepay. While working to build a Change Phone Number feature, we also ran into many "already existing number" problems, due to many users using recycled SIM card in Kenya. Our USSD system which recognizes the caller's phone number was only adding more complexity - a new user with a recycled SIM might be recognized as another existing user, and an existing user who has changed a phone cannot be recognized anymore.

02Clear Head & Think Straight

To make sure that no user is left behind, I went back to the drawing board and laid down all possible and complex scenarios.

03Design for USSD

Although most of us are not used to USSD, it is yet a popular platform that offers a light text-based interactive service in Kenya. It is used to serve not only smartphone users but also feature phone users who are a significant portion of the whole population.
Since the USSD service can recognize the dialer's number, two problems arose.

1. The user was never registered but the phone number is recognized (Recycled SIM card issue).
2. The user is registered, but changed the phone and calling from a new number - thus not recognized.

To crack these cases, I decided to give a new user to claim the recognized number, and a registered user to claim an existing account.
